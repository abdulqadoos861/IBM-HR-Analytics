# 📊 IBM HR Analytics — Employee Attrition Analysis

## 📌 Project Overview
This project focuses on analyzing employee attrition using the IBM HR dataset. The goal is to identify key factors influencing employee turnover and provide actionable insights using data analysis and visualization.

The analysis was performed using Python, covering data cleaning, exploratory data analysis (EDA), and business-driven questions related to workforce behavior.



 📁 Dataset
- Source: IBM HR Analytics Dataset  
- Records: ~1,470 employees  
- Features: Demographics, Job roles, Salary, Satisfaction, Performance, Attrition  

---

⚙️ Project Workflow

 Data Preprocessing
- Removed irrelevant columns (EmployeeCount, Over18, EmployeeNumber, StandardHours)
- Checked missing values and duplicates
- Feature engineering (e.g., Expected Daily Rate)

 Exploratory Data Analysis (EDA)
- Correlation analysis between numerical variables
- Distribution analysis (Age, Income, Satisfaction)
- Department and role-based comparisons



 📊 Key Business Questions Solved

- What is the overall attrition rate?
- Which job roles have the highest attrition?
- Does overtime increase employee turnover?
- How does business travel affect attrition?
- Are younger employees more likely to leave?
- Does salary impact attrition?
- What factors (satisfaction, overtime, performance) drive attrition?



📈 Key Insights

- Employees working overtime have significantly higher attrition (~30%)
- Younger employees show higher turnover rates (~25%)
- Low job satisfaction strongly correlates with higher attrition
- Employees with lower income are more likely to leave
- Frequent business travel increases attrition risk
- Job Level 1 employees have the highest attrition (~26%)
